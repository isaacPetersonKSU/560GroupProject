560GroupProject

Code located in GroupProject Folder. SourceData contains excel sheets used to fill data tables.
PrimaryData folder contains code connecting visual studio code to database. GroupProject/Sql
contains all sql queries including tables, data, schema, and procedures. 

To create database execute the files in GroupProject/Sql in the following order:
1. CreateDatabase.sql
2. Schemas/Season.sql
3. Tables.sql
4. Data/SourceDataForStagingTables.sql
5. Data/Data.sql
6. Procedures/QuestionQueries.sql
7. Procedures/ReportQueries.sql
